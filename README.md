# DATA_221_Assignment_1

This repository contains solutions for Questions 1-8 for Assignment 1 in DATA 221

# Question_1.py
This program multiplies consecutive integers (starting from 1) until the product exceeds a given threshold. The program outputs the final product and the integer that caused the product to exceed the threshold

# Question_2.py
This file contains a function that takes a list of strings and returns a dictionary. The output displays each word along with the length of the word and whether the length is even or odd.

# Question_3.py
This file contains a function that calculates powers and applies it to a list of pairs (base, exponent). Pairs which include negative exponents are skipped and valid results are printed

# Question_4.py
This program generates and sorts a list of random values between 0 and 1, and then finds the first index in which a value is greater than or equal to a randomly generated value of x

# Question_5.py
This file contains a function that compares the area of two circles. The function validates user input, computes the area of the circles, and returns the percentage of the larger circle that is being covered by the smaller one

# Question_6.py
This program does a distribution analysis on a list of numbers. It returns a dictionary showing the percentage of values that are less than or equal to each individual number in the list

# Question_7.py
This file contains a function that converts seconds since midnight into hours, minutes, seconds, and AM/PM format (XX:XX:XX AM/PM)

# Question_8.py
This program uses the pandas library in order to create a Data Frame from given data. A new colum is calculated from exissting columns and the final Data Frame is printed
